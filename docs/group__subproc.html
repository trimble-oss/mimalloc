Detailed Description

A sub-process contains its own arena's and heaps that are fully separate from the main (sub) process. A thread can belong to only one subprocess at a time. This can be used to separate out logically separate parts of a program (like running multiple interpreter instances in CPython).

◆ mi_subproc_id_t

typedef void* mi_subproc_id_t

A process can associate threads with sub-processes.

Sub-processes are isolated and will not reclaim or visit memory from other sub-processes. Each subprocess always has an associated main heap.
